# Draft Prediction Project
このリポジトリは、選手の身体能力データや所属大学などの情報に基づき、ドラフトで指名されるかどうかを予測する機械学習プロジェクトのコードを管理しています。

## 概要

### 目的: 
身体能力テストや大学時代の統計データから、ドラフト指名（Drafted）を予測。

### 主要アルゴリズム: 
LightGBM, XGBoost

### 主な成果: 
Optunaによるハイパーパラメータ最適化と高度な特徴量生成により、高いAUCスコアを達成。
<br>
<br>

## 工夫した点
このプロジェクトでは、単なるモデル作成にとどまらず、予測精度の向上と過学習の抑制のために以下の手法を導入しました。 


### 1. リーケージを防ぐ「OOF Target Encoding」
カテゴリ変数の処理において、単純なターゲットエンコーディング（Target Encoding）を行うとデータリーケージ（正解情報の漏洩）が発生し、過学習の原因となります。


#### 対応策: 
StratifiedKFold を用いた OOF (Out-of-Fold) 処理を実装。

#### 詳細: 
School や Position に対して、各フォールド内で計算された統計量のみを適用し、テストデータには全訓練データの統計量をマッピングすることで、より汎化性能の高い特徴量を生成しました。

#### 独自の算出式: 
特に大学（School）の特徴量では、指名率だけでなく指名数も考慮した独自のスコアリング式 (1 + 指名率)  指名数 を採用しています。


### 2. ドメイン知識に基づく特徴量生成（Feature Engineering）
選手の特性を多角的に捉えるため、生の数値データから以下のインデックスを算出して追加しました。

##### Power Index: ベンチプレスの回数と体重の積（純粋な筋力の指標）

##### Jump Index: 垂直跳びと立ち幅跳びの合算（瞬発力の指標）

##### Speed/Agility Index: 40ヤード走、シャトルラン、3コーンの合算（機敏性の指標）

##### BMI: 身長・体重からの体格指標


### 3. ターゲット情報を活用した交互作用特徴量
ターゲットエンコーディングされたポジション情報（Position_TE）と、実際の身体能力（スピードやBMIなど）を掛け合わせた相互作用特徴量を作成しました
これにより、「そのポジションにおいてどの程度の身体能力が指名に寄与するか」をモデルが学習しやすくなっています。


### 4. Optunaを用いたハイパーパラメータの自動最適化
LightGBMおよびXGBoostの学習において、Optuna を活用してハイパーパラメータを探索しました。


・learning_rate, max_depth, lambda_l1, lambda_l2 などの主要なパラメータを50試行で最適化。


・cross_val_score と StratifiedKFold を組み合わせることで、精度の安定性を確認しながら最適化を進行しました。
